# incr

An Atom package to increment and decrement the number under the cursor.

### Default Keybindings
```cmd-ctrl-k``` to increment  
```cmd-ctrl-j``` to decrement  

### Whatchout!
- Still working on specs
- If you incr .9 it becomes .10
- If you decr .0 it becomes .-1
